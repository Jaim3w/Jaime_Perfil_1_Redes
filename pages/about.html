Pregunta 1

¿Que son los componentes Pasivos en la red?

componentes Pasivos

Los componentes pasivos de una red son aquellos que no requieren energía externa para funcionar y no tienen la capacidad de amplificar o modificar las señales de forma activa.

Pregunta 2

Descripcion tenica de cada componen pasivos

RJ45

El RJ45 (Registered Jack 45) es un tipo de conector de 8 pines utilizado principalmente para cables de red Ethernet en redes de datos. Es el estándar para la mayoría de las conexiones de red en entornos LAN (Redes de Área Local).

Jacks o Keystone

Los jacks o keystone son módulos intercambiables que se instalan en paneles de parcheo o cajas de pared para proporcionar puntos de conexión para cables de red. Pueden ser para diferentes tipos de conectores, incluyendo RJ45, y otros como HDMI

Cable UTP

El cable UTP es un tipo de cable de par trenzado sin blindaje que se utiliza para la transmisión de datos en redes de datos y telecomunicaciones. Está compuesto por pares de cables trenzados para reducir la interferencia electromagnética.

Patch Panel

Un patch panel es un dispositivo de panel de conexiones que centraliza y organiza cables de red en un lugar único. Facilita la gestión y el mantenimiento de las conexiones de red

Pregunta 2DA parte

Descripcion tenica de cada componen pasivos

Patch Cord

Un patch cord (o cable de parcheo) es un cable corto utilizado para conectar dispositivos a un patch panel o a un dispositivo de red como un switch o router. Generalmente se usa para conexiones temporales o para conectar equipos dentro de un rack o sala de servidores.

Fibra Óptica

La fibra óptica es un tipo de cable que transmite datos usando señales de luz a través de hilos de vidrio o plástico. Ofrece alta velocidad de transmisión y gran ancho de banda.

Canaletas

Las canaletas son estructuras de gestión de cables que organizan y protegen los cables de red y otros cables eléctricos. Se instalan en paredes, techos o pisos para mantener los cables ordenados y accesibles.

Pregunta 3

¿Que son los componentes Activos en la red?

componentes Activos

Los componentes activos de una red son aquellos que requieren energía para funcionar y que tienen la capacidad de amplificar, modificar, conmutar o procesar las señales de datos.

Pregunta 4

Descripcion tenica de cada componen activos

Switch

Un switch es un dispositivo de red que conecta varios dispositivos dentro de una red local (LAN) y gestiona el tráfico de datos de manera eficiente. Opera en la capa 2 (enlace de datos) del modelo OSI, utilizando direcciones MAC para dirigir los paquetes a los dispositivos correctos.

Router

Un router es un dispositivo de red que conecta redes diferentes, como una red local (LAN) con Internet. Opera en la capa 3 (red) del modelo OSI y utiliza direcciones IP para enrutar paquetes de datos a través de redes y subredes.

Tarjeta de red

Una tarjeta de red (NIC) es un componente de hardware que permite a un dispositivo (como un ordenador) conectarse a una red. Puede ser para conexiones Ethernet (cableadas) o Wi-Fi (inalámbricas).

Interruptor inteligente WiFi

Un interruptor inteligente Wi-Fi es un dispositivo de control de encendido y apagado que se conecta a la red Wi-Fi de una casa o oficina, permitiendo controlar la electricidad de dispositivos conectados a él a través de una aplicación móvil o comandos de voz.

Access point

Un punto de acceso (access point) es un dispositivo que proporciona conectividad inalámbrica a una red cableada, permitiendo a los dispositivos inalámbricos conectarse a la red local. Opera en la capa 2 (enlace de datos) del modelo OSI y utiliza estándares de red Wi-Fi.

Pregunta 5

Herramientas de cableado

Ponchadora

La ponchadora es una herramienta manual utilizada para conectar cables de red a conectores, generalmente RJ45 o RJ11. Permite insertar los cables en el conector y asegura que los pines del conector hagan contacto con los hilos del cable.

Crimpadora

La crimpadora es una herramienta especializada para crimpar (o fijar) conectores en cables. La función principal es aplicar una presión específica para asegurar los conectores a los cables de manera segura.

Desforradora

La desforradora es una herramienta utilizada para pelar el aislamiento exterior de los cables de red sin dañar los conductores internos. Permite exponer los cables de manera limpia para su conexión a conectores o paneles de conexión.

Tester/Probador de cables

Un tester o probador de cables es una herramienta que se utiliza para verificar la continuidad y el correcto funcionamiento de los cables de red. Permite identificar problemas en el cableado, como cortocircuitos, cables cruzados o roturas.

Pregunta 6

¿Qué es el cableado horizontal?

Cableado horizontal

El cableado horizontal se refiere al cableado que se extiende desde el panel de distribución principal (o sala de telecomunicaciones) hasta los puntos de conexión en las áreas de trabajo o estaciones de usuario, como oficinas, cubículos, salas de conferencias, etc. Es la parte de la red que conecta directamente los dispositivos de usuario final con el sistema de cableado estructurado de la red.
